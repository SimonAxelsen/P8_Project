Welcome to the P8 Project page. This project is a continuation of our previous work in P7, focusing on adaptive media systems and AI-driven interactions.

About the project

Building upon the real-time, locally deployed LLM-driven multi-agent system from P7, this semester we explore further advancements in adaptive multi-party conversational interaction.

Objectives

  • Improve system latency and responsiveness
  • Enhance coordinated turn-taking and non-verbal behaviors
  • Maintain local execution to ensure user privacy
